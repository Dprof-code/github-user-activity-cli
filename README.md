## Overview

Github Activity CLI is a lightweight and efficient command-line tool designed for developers who want to track GitHub user contributions without leaving the terminal. Built with **Node.js**, it fetches real-time data from the GitHub API and provides a clean, summarized view of recent activities, including commits, issues, and repository stars.

## Features

- **Activity Aggregation**: Intelligently groups multiple commits under the same repository for a cleaner output.
- **Real-time Data**: Fetches the latest event data directly from the GitHub REST API.
- **Event Parsing**: Distinctly identifies different event types like `PushEvent`, `WatchEvent`, and more.
- **Minimalist Design**: Zero-bloat CLI interface focused on speed and readability.

## Installation

Follow these steps to set up the project locally on your machine:

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Dprof-code/github-user-activity-cli.git
   ```

2. **Navigate to the Project Directory**

   ```bash
   cd github-user-activity-cli
   ```

3. **Install Dependencies**

   ```bash
   npm install
   ```

4. **Link the CLI Globally (Optional)**
   To run the command from anywhere on your system:
   ```bash
   npm link
   ```

## Usage

Once installed, you can use the CLI to fetch activity for any public GitHub username.

### Basic Command

```bash
github-activity <username>
```

### Example

```bash
github-activity Dprof-code
```

### Help Command

To view the help documentation and available formats:

```bash
node bin/index.js --help
```

**Output Example:**

- Pushed 3 commits to Dprof-code/github-user-activity-cli
- Opened a new issue in Dprof-code/project-x
- Starred facebook/react
